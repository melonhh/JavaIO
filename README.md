# JavaIO
Java io模型与nio，aio基础

### File类 --- java.io.File
实现Serializable，Comparable两大接口，可以实现：
1. 创建一个文件/文件夹
2. 删除文件/文件夹
3. 获取文件/文件夹
4. 判断文件/文件夹是否存在
5. 对文件夹进行遍历
6. 获取文件信息

File类的静态属性
1. 路径分隔符：windows --- ;  Linux --- :
2. 名称分隔符：windows --- \  Linux --- /

File类的构造方法
* 单参数 --- 绝对路径/相对路径都行
* 双参数 --- 父 ， 子




在Java程序中获取根路径的方法：GetRootPath.java
